WebRTC demos

This repository is for WebRTC demos.

Many of these examples use new browser features. They may only work in Chrome Canary and/or Firefox Beta, and may require flags to be set. Some examples may be retained here, though no longer functional.

All the examples that use getUserMedia() must be run from a server or from localhost. Calling getUserMedia() from a file:// URL will result in a PERMISSION_DENIED NavigatorUserMediaError.

All of these examples use adapter.js, a shim to insulate apps from spec changes and prefix differences. In fact, the standards and protocols used for WebRTC implementations are highly stable, and there are only a few prefixed names. For full interop information, see webrtc.org/interop.

For more information about WebRTC, we maintain a list of WebRTC Resources.

If you've never worked with WebRTC, we recommend you start with the 2013 Google I/O WebRTC presentation.
